README
======

You can download the Prim8Mobile APK Android installation file from _www.prim8software.com_ or compile the software according to the current Android APK. You may also want to download templates for importing your Individuals and Behaviors (Ethogram) files.  Please note: You should save these templates with the given filenames. However, you MUST name the your final Individuals and Behaviors import files as follows: _prim8_import_biosamples.csv_, _prim8_import_ethogram.csv_ and _prim8_import_individuals.csv_ respectively.

What is this repository for?
-----------------------------

Prim8 Mobile is a freely-available Android OS based application that provides a flexible, customizable, and efficient method to collect behavioral field data in both populated and remote areas.
